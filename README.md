##Proyecto
Es una plataforma para buscar compañeros de viaje en coche.

##Tutorial de instalación
```
  $ git clone git@github.com:Losnen/Proyecto-UYA.git
  $ npm install
  $ node app.js
```

##Tecnologías usadas:
* Node JS
* Express
* Materialize CSS
* JQuery
